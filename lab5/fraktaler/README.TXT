PROJECT TITLE: Fraktaler
PURPOSE OF PROJECT: To add three fractals to a selection of fractals that the user can paint on a canvas.
VERSION or DATE: 11 May 2017
HOW TO START THIS PROJECT: Run the main method
AUTHORS: Fressia Merino & Therese Tengdahl
USER INSTRUCTIONS: Choose what fractal you want to paint on the canvas, then put a number in the box for how many levels you want the fractal to go. Click on "Trace on" if you want to see how the fractals are painted.
